【断更】接下来的训练计划

2016-08-16 OI/ACM 0 Comments

要做的事

2016.9.1开始的一个月：

  1. 以学校的模拟赛题、知识点为主要任务。在时间有限的情况下优先该条。
  2. 周二、四花时间做coci。目标是在9月完成首页后两套题，并记录好的脑洞。
  3. 周一、三、五花时间做专题。按照目录进行。
  4. 每周随机一个oj（JZOJ、bzoj、CF、计蒜客），在上面随机至少1个题，不论难度、代码量，并记录好的脑洞。（题目若认定为普及组难度则继续随机）

2016.10.1开始的一个月：

  1. 以学校的模拟赛题、知识点为主要任务。在时间有限的情况下优先该条。
  2. 周二、四花时间做coci。目标是在10月完成2015~2016的#5和#6，并记录好的脑洞。
  3. 周一、三、五花时间做专题。按照目录进行。
  4. 每周一题，然而不强求。

2016.10.31~noip2016：

  1. 以学校的模拟赛题、知识点为主要任务。在时间有限的情况下优先该条。
  2. 学已讲专题，做自己的专题
  3. noip的重难题、易错题，有价值的真题去做
  4. 手边时刻放一道可思考的题，可以来自coci、jzoj。

2016.12.1开始的一个月：

  1. 以学校的模拟赛题、知识点为主要任务。在时间有限的情况下优先该条。
  2. 学习专题，主要为安排的省赛专题，去实现高级算法模板，并至少解决课件例题。
  3. oj开始填坑，随机选择标了星的题来做。该项在数论专题讲完后开始实行。
  4. 做coci、cf、bc的比赛！！！

2017.1.1开始的一个月：

  1. 模拟赛改题
  2. 专题以模拟赛题涉及的为主，比如最近的博弈
  3. oj填坑，随机选择标了星的题来做。
  4. 做各种高质量网络赛

2017.1.22~1.31：

  1. wc模拟题以及相应的A组题
  2. 写待写模板，以及把最近模拟赛涉及到的知识点写一写
  3. 开两场cf练习赛，每场从C开始做
  4. 春节hackerrank、26号uoj

2017.2.10开始的一个多星期

  1. 认真对待每一场koi模拟，踏踏实实改题，寻找稳的方法。
  2. 过一遍字符串、数据结构课件
  3. 有空的话研究一下WC课件
  4. 好好考gdkoi

2017.2.26开始的一个月

  1. 改题，欠下的太多了
  2. 过字符串、几何、博弈课件
  3. 思考题来源：cf、bc、atcoder、集训队作业、杂题选讲课件、coci
  4. 好好考浙江省选

2017.3.26开始的一个月

  1. 改题
  2. 专题复习
  3. 思考题来源：cf、集训队作业、coci（至少两套）、去年湖南集训
  4. 好好考江苏、GDOI模拟

2017.4.10~2017.4.26

  1. GDOI 模拟*4
  2. atcoder樊爷题集、coci 继续，并记录脑洞和扩展
  3. 过大家的专题复习课件，在 23 号前完成，若有多余时间则过专题讲课课件；23 号后去浏览大神博客里的好题、学习小记、总结

2017.5.3~2017.5.21

  1. 专攻智商题，收集共享上所有杂题选讲课件，以及浏览 jasonvictoryan 的博客。
  2. 补知识点。字符串（SAM为主）->数据结构（平衡树LCT为主）->多项式（生成函数为主）
  3. 参与网上的比赛（codejam、cf、bc等）

2017.5.31~2017.7.19

  1. noi 模拟题
  2. 实现或精通一些未熟悉的算法（洲阁筛、SAM、ZKW、LCT、treap、矩阵树定理、几何、高次剩余那套理论……）
